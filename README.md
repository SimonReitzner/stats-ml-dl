### Usage <a id="usage"></a>
1. Clone the repository
   ```sh
   git clone https://github.com/SimonReitzner/stats-ml-dl.git
   ```
2. Access subfolder
   ```sh
   cd stats-ml-dl
   ```
3. Get data
   ```sh
   python main_get_data.py
   ```
